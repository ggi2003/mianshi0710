# WORLDVIEW 监控软件界面布局拆解

整体界面为**深色军工风全息雷达监视系统 UI**，以纯黑色为底色，核心视觉是居中圆形全域态势监视主画面，页面分为**顶部标题区、左侧信息面板、中央主监视视窗、右侧参数面板、底部功能控制栏**五大区域，整体像素颗粒复古监控屏质感。

## 一、左上角：软件标识与密级文档信息区

1. 最顶端大字标题：**WORLDVIEW**，为软件系统名称；右侧小字标注 `PLAYBACK`（回放模式按钮），代表当前处于录像回放查看状态。
2. 标题下方是涉密文档页眉信息：
   - 密级标注：`TOP SECRET // SI-TK // NOFORN`（最高机密、仅限特定渠道、禁止对外分发）
   - 编号串：`KB11-6040 OPS-4138`
   - 状态标签：大号 `NORMAL`，下方小字 `NORMAL GLOBA`（全局正常模式）
3. 再往下是 3 个竖向排列的折叠式侧边信息输入框，无展开内容，属于左侧附属信息栏。

## 二、中央核心：圆形全域态势主监视窗口（界面视觉中心）

页面正中是**大圆形雷达 / 卫星态势监视画面**，是软件核心显示区域：

1. 底图：叠加中东波斯湾区域卫星地貌底图，陆地为土黄色荒漠纹理，海域填充深蓝色；
2. 地理标注：画面内标注关键地名：**BAHRAIN（巴林）**、**QATAR（卡塔尔）**、**UNITED ARAB EMIRATES（阿联酋）**，清晰框定波斯湾空域海域范围；
3. 态势要素：
   - 大量浅蓝色线条 + 飞机图标：代表**民航 / 军用航线、实时航空器航迹**，布满波斯湾上空；
   - 红色折线边界线：为划定的禁飞区 / 任务隔离边界；
   - 黄、红色高亮标记点：代表重点目标、告警点位；
4. 画面左下角附带经纬度坐标：`22°14'46.55"N 058°51'11.71"E`，锁定当前监视锚点地理位置；
5. 画面底部居中悬浮小弹窗：`Location`，为当前定位提示框。

## 三、右上角：系统录制与分级参数面板

1. 最上方录制日志：`REC 2026-03-03 01:03:33Z`，带日志编号与 DESC 备注，记录回放片段的 UTC 录制时间；
2. 下方 3 组竖向功能模块，每组包含标题、指示灯、数值参数：
   - 第一组：`FORCE`，蓝色指示灯 + 数字参数；
   - 第二组：`SURVEILLANCE`，蓝色指示灯 + 数字参数；
   - 第三组：`THREAT`，下拉框可选`Tactical`（战术级威胁）；
3. 面板最底端两行设备遥测数据：
   - `GSO: 575.68M NTIRS: 0.0`
   - `ALT: 1935159N SUN: -24.3° EL`
   用于显示卫星轨道、高度、太阳俯仰角等传感设备参数。

## 四、左下角：坐标与底层状态码

圆形主画面左下方，在黑色背景上单列一行设备回传报文：
`MSGS: 39Q VE 9544 5824`，为系统消息队列、设备硬件状态编码。

## 五、底部通栏：全局功能控制工具栏（界面最下方整行）

横向排布多档位速度控制、功能开关、分类图层勾选按钮，从左至右：

1. **回放倍速控制**：`1/6x、1/3x、1x、5/x、15/x、1h/x` 多级回放速度档位，搭配`PAUSE（暂停）`、`OFF`开关；
2. **视角与传感参数**：`AZIM（方位角）`、`ELEV（俯仰角）`角度调节滑块；
3. **功能快捷按钮**：`ALERT（告警）`、`SPIRAL OUT（螺旋外扩扫描）`、`250km`监视距离阈值、`45°/60° FOV（视场角）`；
4. **图层勾选分类（带图例颜色）**：
   - ✅`Commercial Flights`民航航班（白色标识）
   - ✅`Military Flights`军用飞行器（黄色标识）
   - `Kinetic`动能打击、`Retaliation`反击行动、`Civilian Impact`民用影响、`Maritime`海上态势、`Infrastructure`基础设施、`Escalation`事态升级、`Airspace Closure`空域封锁；
5. **专项任务模块按钮**：`Routine Traffic`常规交通、`Airspace Classify`空域分级、`VIP Intercept`要员拦截、`Internet Blackout`断网监测、`Country Boundaries`国界线、`OSINT Social Events`开源情报社交事件。

## 六、整体交互视觉特征

1. 全局采用暗底荧光青蓝色电子屏 UI，模拟军用卫星情报监控终端；
2. 信息层级：**中央地图主视图 > 上下左右参数面板 > 底部全局控制栏**，符合指挥监视类软件「一图总览 + 周边参数 + 底部操作」经典布局逻辑；
3. 右上角与左上角固定承载元数据（密级、时间、设备参数），底部作为所有图层、回放、模式的总操控入口。
